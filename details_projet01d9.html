Outils de Forensique

Informations de Développement

  • Catégorie: Hacking Éthique
  • Technologies: Python

Description :

Mon projet de Forensique en Python est un outil puissant qui permet de récupérer des informations précieuses à partir de différents types de fichiers. Il peut extraire les métadonnées d'un fichier PDF, découvrir du texte caché dans les programmes, révéler les métadonnées EXIF présentes dans les images, trouver des coordonnées GPS et détecter les artefacts de Firefox.
