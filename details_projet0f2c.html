Scanner de Vulnérabilités Web

Informations de Développement

  • Catégorie: Hacking Éthique
  • Technologies: Python

Description :

Mon scanner de vulnérabilité web est mon plus gros projet Hacking éthique en Python. Il consiste à créer un scanner avec une interface graphique afin d'analyser une URL, scraper le site web et rechercher plusieurs failles (SQL & XSS). J'ai utilisé plusieurs modules comme Beautiful Soup, Requests, Tkinter ou Threading. Comme bonus, à la fin du scan mon programme génère un rapport HTML.
