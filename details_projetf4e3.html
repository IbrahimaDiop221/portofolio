DNS & ARP Spoofer

Informations de Développement

  • Catégorie: Hacking Éthique
  • Technologies: Python

Description :

Ce projet consiste en la création d'un renifleur de réseau qui permet de récupérer les mots de passe en clair. Le DNS Spoofer, grâce aux modules Iptables et Scrapy me permet d'intercepter le trafic sur le réseau et modifier en temps réel une page web. Enfin, mon ARP Spoofer permet de détourner les données en modifiant les associations IP/MAC dans le réseau.
